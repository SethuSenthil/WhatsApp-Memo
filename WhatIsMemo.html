What is Whatsapp Memo?

Whatsapp Memo is a program that generates a link which allows you to unlock a incredible feature. This feature allows the user to text themselves in whatsapp , without creating a group with only one user. Instead Whatsapp Memo does this in a more straight-forward and elegant manner which actually texts the user and not a group.

Ok, What is so special about texting myself?

Whatsapp is one of the most daily used apps in the world, so wouldn't it be more convenient to keep your thoughts, shopping lists, notes and a whole lot more in the same place in one app you always craving to open?
